## Design Philosophy
MISTCODER is not a rule-based scanner. It is a reasoning system that models
software behavior, constructs adversarial mental models, and discovers
vulnerability chains beyond human intuition.

## Core Properties
- Language-agnostic ingestion
- Parallel static + dynamic analysis
- Self-improving detection models
- Human-governed simulation execution
