# MOD-02 -- Static & Dynamic Analysis Engine

Performs deep code analysis across two dimensions simultaneously.

## Static Analysis
- AST parsing and taint flow tracing
- Data flow and control flow graph construction
- Dependency vulnerability graphing (transitive CVE detection)

## Dynamic Analysis
- Sandboxed execution and runtime behavior monitoring
- Memory anomaly detection
- API call tracing and anomaly flagging
